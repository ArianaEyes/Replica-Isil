# Pagina-ISIL

## Descripción

Aplicación web cuya interfaz replica el diseño público de ISIL+ con fines educativos y demostrativos.

El proyecto implementa una arquitectura full stack que contiene frontend y backend, permitiendo la gestión dinámica de contenido mediante operaciones CRUD (Create, Read, Update, Delete). Esto facilita la actualización y mantenimiento de los componentes sin modificar directamente el código estático.

## Tecnologías

- Angular (Frontend)
- Java (Backend)
- MSSQL Server(Base de datos)
- REST API
- JSON

## Arquitectura

- Cliente SPA desarrollado en Angular.
- API REST construida en Java.
- Persistencia de datos en MSSQL Server.
- Comunicación mediante HTTP y formato JSON.

## Deployment

- **Frontend: Vercel**
- Backend: Render

## Instalación

npm install

## Ejecución

npm start

## Funcionalidades

- Sistema de autenticación.
- Gestión de registros mediante CRUD.
- Conexión segura a base de datos.
- Separación por capas (controlador, servicio, persistencia).
